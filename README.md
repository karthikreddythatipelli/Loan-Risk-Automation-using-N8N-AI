# Loan-Risk-Automation-using-N8N-AI

AI Loan Risk Analysis System

## Overview
This project is an *AI-powered loan risk analysis and automation system* built using *n8n workflows and Generative AI*.

It enables:
- Loan-level risk explanation (Q&A)
- Portfolio-level risk summary
- Automated email reporting

The system transforms raw loan data into *clear, business-friendly insights*.

## Key Features

### 🔹 1. Loan-Level AI Q&A
- Accepts user queries via Webhook
- Supports questions like:
  - “Why is this loan risky?”
  - “Explain loan 6297794”
- Extracts key loan attributes:
  - Grade
  - Interest Rate
  - Income
  - Purpose
- Generates *short, explainable AI responses*



### 🔹 2. Portfolio Risk Summary
- Analyzes entire loan dataset
- Provides:
  - Overall risk level
  - Key risk drivers
  - High-risk regions
  - Business recommendations
 
### 🔹 3. Automated Email Reporting
- Uses Gmail OAuth2 integration
- Sends AI-generated insights automatically
- Useful for risk teams and leadership updates

## ⚙️ Tech Stack

- *n8n* – Workflow automation  
- *OpenAI / OpenRouter (LLM)* – AI reasoning  
- *Excel Dataset* – Loan data  
- *Webhook API* – Input/output  
- *Gmail API (OAuth2)* – Email automation  
- *Power BI* – Dashboard visualization
